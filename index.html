Watchman-Route-Optimal
This project was involved in implementing minimum Security Cameras given a simple environment. A simple environment is something that is realistic, like in real world. This project involved reading a lot of research papers, and implementing them in Python 3. We plotted the graphs using Matplotlib. We then used these points, and created a logic, that if these points are optimal then if a guard robot passes these points, then it will create a watchman route in shortest time and path. The logic involved using Travelling Salesman Problem and adding some constrains to the guard, such as "how far should the guard be from the edge to consider it guarded?".

rbe500-pid-project
This was the first project that I did in my MS during robotics. We were tasked with designing and simulating PID controller over a RRP robot. We had to use ROS to simulate this in Gazebo which is a real world physics simulation application. We created a server, to which the joint values were published and subscribed. We created and implemented inverse kinematics for the robot to track the trajectories of joints to reach the given desired locations.
