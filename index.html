Watchman-Route-Optimal
This was our project in Path Planning for a 2D Robot. Given an area of the environment of simple environment, we found locations of camera locations such that all the edges of environment are guarded at all times. We added some constraints to how much degrees it spans and its length. Using these locations, we tried to construct a watchman route. A watchman route is the route which a mobile security guard can take to visit every single edge of the given environment in the shortest amount of time and route. We constructed these routes using Travelling Salesman Problem. The route is "optimal" but only in the context of discretized world. The more we discretize the environment, the better result we will get.
